- 1️⃣ What is the difference between var, let, and const?
- Ans: Var means variable. We can change its value. Let is same to var and const means constant that value never be changed! In var, we can change the value several times in var but not in let. 
- 2️⃣ What is the spread operator (...)?
- Ans: It takes all items out of an array and spreads them into a new context.
- 3️⃣ What is the difference between map(), filter(), and forEach()?
- map is used when we want to change anything into an array and get a new array. filter() is used to get matched value from data and forEach() is also a loop function that loop and search. Basically Map(), Filter() and forEach all are like a loop!
